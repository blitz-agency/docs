The **Company Search** endpoint (`POST /v2/search/companies`) lets you find companies matching precise criteria. Use it to build ABM lists, identify ICP-matching accounts, or power dynamic prospecting workflows.

<Info>
  **Cost**: 1 Credit per result returned. If `max_results` is set to 25 and the query matches 25+ companies, the request costs 25 credits.
</Info>

## How It Works

You send a `POST` request with a `company` object containing your filters. All filters are optional and combined with **AND** logic. Within each filter, multiple values use **OR** logic.

The API returns a paginated list of company profiles matching your criteria.

```json
POST /v2/search/companies

{
  "company": {
    "keywords": { "include": ["SaaS", "cloud platform"] },
    "industry": { "include": ["Software Development"] },
    "hq": { "continent": ["Europe"] },
    "employee_range": ["51-200", "201-500"]
  },
  "max_results": 25
}
```

## Filter Reference

### `name` — Company Name

Search for keywords in the company name.

```json
"name": {
  "include": ["Stripe", "Adyen"],
  "exclude": ["Holdings"]
}
```

### `keywords` — Description Search

Search for keywords in the company's LinkedIn description/about section.

```json
"keywords": {
  "include": ["AI", "machine learning"],
  "exclude": ["consulting"]
}
```

### `industry` — Industry Filter

Filter by LinkedIn-normalized industry labels. These are **exact match** values.

```json
"industry": {
  "include": ["Software Development", "Financial Services"],
  "exclude": ["Staffing and Recruiting"]
}
```

<Warning>
  Industry values must match the exact LinkedIn taxonomy. A typo or variation (e.g., `"SaaS"` instead of `"Software Development"`) will return 0 results. See the [full list of accepted industries](/guide/reference/field-normalization#industries).
</Warning>

### `hq` — Headquarters Location

Filter by the company's headquarters. Combine sub-fields for precision.

```json
"hq": {
  "continent": ["Europe"],
  "country_code": ["FR", "DE", "GB"],
  "sales_region": ["EMEA"],
  "city": { "include": ["Paris", "Berlin"] }
}
```

<Tip>
  Use `sales_region` for broad GTM targeting (NORAM, LATAM, EMEA, APAC). Use `country_code` or `city` for surgical precision. See [Field Normalization](/guide/reference/field-normalization) for all accepted values.
</Tip>

### `employee_count` — Exact Range

Set a numeric range on the actual LinkedIn employee count.

```json
"employee_count": { "min": 100, "max": 500 }
```

### `employee_range` — LinkedIn Buckets

Use LinkedIn's predefined size buckets. Pass multiple values for OR logic.

```json
"employee_range": ["51-200", "201-500", "501-1000"]
```

Accepted values: `1-10`, `11-50`, `51-200`, `201-500`, `501-1000`, `1001-5000`, `5001-10000`, `10001+`.

### `founded_year` — Founding Year Range

Target startups or established companies by founding year.

```json
"founded_year": { "min": 2018, "max": 2024 }
```

### `type` — Company Type

Filter by LinkedIn company type.

```json
"type": {
  "include": ["Privately Held", "Public Company"],
  "exclude": ["Government Agency"]
}
```

Accepted values: `Educational`, `Educational Institution`, `Government Agency`, `Nonprofit`, `Partnership`, `Privately Held`, `Public Company`, `Self-Employed`, `Self-Owned`, `Sole Proprietorship`.

### `min_linkedin_followers`

Set a minimum follower threshold to filter out shell or inactive pages.

```json
"min_linkedin_followers": 500
```

## Pagination

The API supports cursor-based pagination. Each response includes a `cursor` field. Pass it in the next request to get the following page.

```json
// First request
{ "company": { ... }, "max_results": 25 }

// Response includes: "cursor": "eyJwYWdl..."

// Next page
{ "company": { ... }, "cursor": "eyJwYWdl...", "max_results": 25 }
```

When `cursor` is `null` in the response, you've reached the last page.

## Recipes

### The "ICP Builder" — SaaS Scale-ups in Europe

Find mid-size SaaS companies across Europe for an ABM campaign.

```json
{
  "company": {
    "keywords": { "include": ["SaaS", "platform", "cloud"] },
    "industry": {
      "include": ["Software Development", "Technology; Information and Internet"],
      "exclude": ["Staffing and Recruiting", "Management Consulting"]
    },
    "hq": { "continent": ["Europe"] },
    "employee_range": ["51-200", "201-500"],
    "type": { "include": ["Privately Held"] },
    "min_linkedin_followers": 200
  },
  "max_results": 50
}
```

### The "Startup Hunter" — Early-stage FinTech in the US

Identify recently-founded FinTech companies for partnership opportunities.

```json
{
  "company": {
    "keywords": { "include": ["fintech", "payments", "neobank", "banking API"] },
    "hq": { "country_code": ["US"] },
    "founded_year": { "min": 2021 },
    "employee_range": ["1-10", "11-50"],
    "type": { "include": ["Privately Held"] }
  },
  "max_results": 25
}
```

### The "Enterprise Mapper" — Large companies in APAC

Map large public companies across the Asia-Pacific region.

```json
{
  "company": {
    "hq": { "sales_region": ["APAC"] },
    "employee_range": ["5001-10000", "10001+"],
    "type": { "include": ["Public Company"] },
    "min_linkedin_followers": 10000
  },
  "max_results": 50
}
```

## Combine with Waterfall Search

Company Search becomes extremely powerful when chained with the **Waterfall ICP Search**. Use Company Search to build your target account list, then pipe each `linkedin_url` into the Waterfall to find decision-makers.

```
Company Search → Get linkedin_url → Waterfall ICP → Get decision-makers → Email Enrichment
```
