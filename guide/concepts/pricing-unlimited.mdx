<Note>
  **Free Trial**: Every new account gets **1,000 free credits** to test the API before subscribing. Credits are used only on demo accounts. All paid plans are fully unlimited.
</Note>

## Rate Limits

All plans include the same technical rate limit:

| Metric | Value |
| :----- | :---- |
| **Requests per second** | 5 RPS (standard) |
| **Concurrency** | 5 simultaneous requests |
| **Timeout recommendation** | 10s (search & enrichment), 20s (email validation) |

<Tip>
  Your exact rate limit is returned by the `GET /v2/account/key-info` endpoint in the `max_requests_per_seconds` field. Use this value to configure your client-side rate limiter.
</Tip>
